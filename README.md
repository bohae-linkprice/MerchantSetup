## 머천트 셋업

제휴마케팅에서 광고주가 돈을 지불하는 성과는 어필리에이트 사이트 (또는 페이지) 방문자가 광고 배너 또는 링크를 클릭하여 머천트 사이트로 이동, 약속된 행위를 하는 것입니다. 

성과를 정확히 측정하기 위해서는 어필리에이트 사이트 방문자의 이동경로와 머천트 사이트에서의 
구매(가입)에 대해 추적이 가능해야 하며, 이를 위해 필요한 것이 머천트 셋업입니다.
